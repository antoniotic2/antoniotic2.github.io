Enlaces con textos e imágenes

Para hacer un enlace que nos lleve a otra página tenemos que escribir dentro del comando en que queramos hacer el enlace, por ejemplo dentro de <p>...</p> si queremos que el enlace sea un texto o dentro de <img> si es una imagen. A continuación, escribimos <a>...</a>. Todo lo que este dentro de este comando formará parte del enlace. Si hay algo fuera de este comando, pero dentro del anterior no formará parte del enlace. Para establecer la página a la que queremos que nos lleve escribimos href="" dentro de <a>. Dentro de las comillas escribimos la dirección de la página.

Enlaces absolutos y relativos

  • Enlace absoluto: En este tipo de enlace dentro de href="" escribimos la url de la página. Se suele usar para llevarnos a páginas fuera de nuestro sitio web.
    Por ejemplo: Pincha en la imagen para ir a la página de rtve.
  • Enlace relativo: En este caso escribimos la ruta que tiene que seguir dentro de nuestra carpeta raíz. Por tanto se usa para págians que están dentro de nuestro dominio.
    Por ejemplo: Pincha aquípara volver a la página de inicio.

Enlaces externos e internos

  • Enlace externo: Lo usamos para llevarnos a una página distinta, ya sea de nuestro dominio o no. Todos lo ejemplos anteriores son de este tipo, de manera que solo tenemos que elegir uno de ellos para hacer un enlace externo.
  • Enlace interno: Lo usamos para dirigirnos a algun punto de la página en que nos encontremos en ese momento. Para hacer un enlace interno tenemos que dar una ip al texto al que nos dirigiremos con el enlance (pincha aquí para ver como usar la ip). A continuación, dentro del enlace, en href="", dentro de las comillas, escribimos #nombre_ip.
