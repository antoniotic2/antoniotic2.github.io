INTRODUCCIÓN A CSS

Seleccionar por nombre de etiqueta

Para ello escribimos, dentro de <head>, el comando <style> y lo cerramos con </style> y a su vez dentro de este escribimos el nombre de la etiqueta que queramos modificar seguida de los símbolos {}, dentro de los cuales escribimos la propiedad que queremos modificar. Estas son algunas de ellas:

  • "color:" y seguido del nombre del color o de rgb(x,y,z), donde x,y,z son los números correspondientes al color deseado. Con esto cambiamos el color del texto.
  • "background-color:" y seguido del nombre del color o de rgb(x,y,z). Esto permite cambiar el color del fondo de ese texto.
He usado esto para el título de la página y de los diferentes apartados.


Seleccionar por id

Para ello asignamos una ip al comando que queramos escribiendo id="" dentro del comando y le otorgamos un nombre único.

A continuación, dentro del comando <style> escribimos #nombre_de_la_ip {}. Dentro de las llaves hacemos las modificaciones.

Este es un ejemplo usando la ip


Seleccionar por clase

Lo utilizamos para modificar varios elementos sin modificar todos los del mismo tipo de la página web y sin tener que adjudicar una ip a cada elemento. Para ello escribimos dentro del comando class="" e introducimos el mismo nombre en cada elemento que queramos que tenga las mismas propiedades dentro de las comillas.

Ahora, de nuevo, dentro de <style> escribimos ".nombre_de_la_clase" y cambiamos las propiedades que queramos.

He usado esto para modificar estos tres párrafos.
