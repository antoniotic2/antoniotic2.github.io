CSS: Propiedades del texto


Familia de fuente genérica y específica

Las familias de fuente las usamos para cambiar el tipo de letra de nuestra página web. Para ello, dentro de la etiqueta <style> escribimos dentro del tipo de de etiqueta a la que queramos cambiar su fuente font-family:;.

  • Específica: la usamos para establecer un tipo de fuente en concreto. La escribimos después de ":", y a continuación escribimos "," para poner a continuación la fuente genérica.

    Un ejemplo de este tipo de fuente es Verdana.

  • Genérica: esta fuente establece una fuente que será diferente según la fuente predeterminada que tenga nuestro navegador.

    Algunos ejemplos de fuentes genéricas son sans-serif y serif

Tamaño del texto

Para cambiar el tamaño del texto usamos la propiedad font-size: y escribimos el tamaño que queremos que tenga acompañado de las abreviaturas de las unidades. Algunas unidades son los pixeles (px) y em que se utiliza para establecer un tamaño respecto al tamaño predeterminado del texto de los párrafos, el cual corresponde a 1em.
