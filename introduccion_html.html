INTRODUCCIÓN A HTML

HTML boilerplate

Es la estructurta de toda página web. En el caso de Visual Studio Code lo hacemos aparecer escrbiendo html. Después pinchamos en html:5.


Encabezados, de h1 a h6

Los usamos para destacar texto por su tamaño. Para escribir con un tipo de encabezado escribimos el texto dentro del comando <h1>(para abrir el comando), u otro número del 1 al 6, y </h1>(para cerrar el comando).

Salto de línea con br


Aquí hemos usado <br> para dejar un espacio en blanco. He usado esto mismo para dejar espacio entre los diferentes apartados.



Cursiva

Para destacar lo que queramos con cursiva lo escribimos entre <em> y </em>.

Donde se vea algo en cursiva


Negrita

Para destacar lo más importante usamos la negrita. Para ello escribimos lo que queramos poner en negrita entre el comando <strong> y </strong>.

Lista no ordenada

La usamos para hacer una lista de viñetas. Para ello escribimos <ul> y </ul>, y dentro escribimos <li> y </li> para cada elemento de la lista.

Lista ordenada

Con ella podemos hacer una lista enumerada escribiendo el comando <ol> y </ol>, y dentro de este el comando <li> y </li> para cada elemento de la lista, al igual que con la lista no ordenada.

Insertar imagenes

Para insertar imágenes en nuestra web tenemos que usar el comando <img>. En este caso no hay que poner comando de cierre. Dentro de este podemos escribir otros comandos que irán seguidos de un igual y entre comillas escribimos el contenido correspondiente a cada comando. Algunos de ellos son:

  • src: entre las comillas escribimos la ruta de la carpeta en que se encuentra la imagen, seguido de "/" y del nombre de la imagen.
  • alt: aquí escribimos el texto que queramos que aparezca en caso de que no se pueda cargar la imágen.
  • width: en este caso debemos introducir un número que corresponde al número de pixeles de ancho que va a ocupar la imagen.
